Bookmark Facility Component

 

 

Introduction:

Consist in a simple form with a list box to add, delete and select bookmarks related to the different forms of the system.

 

 

 

 

 

Parts of the Component:

  • frmBookmarks
  • tblAuxBookmarks (this table should be place in sql)
  • modBookmark
  • modADO_DAO

 

 

How to use it:

1- After you decide in witch form you want to have the bookmark facility add a new case statement for this form into the Form Load event of the frmBookmarks with a sql that identify the key field of the form:

 

            Case c_Forms_frmContactsRecord

                strSQL = "SELECT FirstName FROM tblContacts " & _

                         "WHERE IdContacts = " & m_lngMainFieldId

                lblMainValue.Caption = "Contact:"

 

 

2- Call the form from the event you want to fire the bookmark

 

 

‘Replace the main field and the form name in the openagr argument

DoCmd.OpenForm "frmBookmarks", acNormal, , , , acDialog, Me.IdCountries & "|" & c_Forms_frmDemo

 

If p_lng_frmSearchId = 0 Then

    Exit Sub

Else

    proFindRecord p_lng_frmSearchId

End If

 

3- The selected value will be in the p_lng_frmSearchId you can use it however you want.

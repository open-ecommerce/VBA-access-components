IUD (Insert, Update, Delete) Main Container

 

 

Introduction:

The main objective of this component is to reuse the generic functionality required in most of the forms. 

The component is based in a form with a sub form control where the system forms are load.

There is also an horizontal menu below with a series of command buttons related with generic public functions such as Insert, Delete, Search, Bookmark among others.

 

 

 

 

Parts of the Component:

  • frmTemplateMain
  • modTemplate
  • frmSearch
  • modSearches
  • frmHelp
  • frmBookmarks
  • modFunctions
  • modConVars
  • clsErr

How to use it:

1- The first thing is create the forms you want to be contain into the frmTemplate you can have 3 different type of forms if you only want 1 type you just have to hide the commands that call the types you don’t want to display.

 

As good practices the 3 forms can be call Record, List and tree so if you have a contact form they will be frmContactsRecord, frmContactsList and frmContactTree.

 

You can create the form how ever you want and can have other chills related but the main form have to be bounded with a recordset so the property Record Source have to be fill.

 

To relate the forms with the frmTemplate you also have to add the following lines into the current event of the form.

 

    Me.Form.Parent.frmCurrentId = Nz(Me.IdContacts, 0) ‘replace the IdContacts with your main id

    Me.Form.Parent.proShowRecords

 

 

2- Add the a generic name for the form used by the template in the enum emn_TemplateBasedForms

'List of forms using instance that is necessary to know

'in with instance of the form we are working on

Public Enum emn_TemplateBasedForms

    Contacts

    Search

End Enum

 

3- Add a case for enum created into p_funLoadTemplate with the functionality you want for the template

 

            Case emn_TemplateBasedForms.Contacts

                .subMainForm.SourceObject = c_Forms_frmContactsRecord

                .Caption = "Contacts Data Base"

                .lblTitle.Caption = "Contacts Datbase"

                .lblFilterTitle.Caption = "Filter by Region"

                .frmBrowsingModeForm = c_Forms_frmContactsList

                .frmRecordModeForm = c_Forms_frmContactsRecord

                .frmTreeModeForm = c_Forms_frmContactsTree

                .frmIdField = "IdContacts"

                .frmSearch_Value = emn_SearchWithForm.Contacts

                .cmdLeft_cmd.Visible = True

                .cmdLeft_cmd.Caption = "Show Cities"

                .imgHelp.Visible = True

                .imgSendMail.Visible = True

                .frmHelpFile = Application.CurrentProject.Path & c_Help_Files_Path & c_Help_IUD_MainContainer

                .cmdDeleteRecord.Visible = True

                .cmdBookmark.Visible = True

                .cmdModeTree.Visible = True

 

4- If you going to use the Insert and Delete functionality, add a new case with the correct sql strings in the following functions:

       

p_pro_AddNewRecord:

Case emn_TemplateBasedForms.Contacts

            strMsg = "Are you sure, you want to create a New Contact?"

           

            intAnswer = MsgBox(strMsg, vbYesNo)

            If Nz(intAnswer, 9) = vbYes Then

                           

                strSQL = "INSERT INTO tblContacts " & _

                                    "(LastName)" & _

                                "VALUES " & _

                                    "('name');"

               

                DoCmd.SetWarnings False

                DoCmd.RunSQL strSQL

                DoCmd.SetWarnings True

            End If

 

p_fun_DeleteRecord:

Case emn_TemplateBasedForms.Contacts

             strSQL = "DELETE * FROM tblContacts Where IdContacts = " & lngCurrentId

             DoCmd.SetWarnings False

             DoCmd.RunSQL strSQL

             DoCmd.SetWarnings True

             p_funDeleteRecord = True

 

5- To call the form use the following statement in the event you choose:

 

               p_funLoadTemplate emn_TemplateBasedForms.Contacts

 

      You can also call the template from the search

    lngId = p_funSearchWithForm(emn_SearchWithForm.Contacts)

    If lngId = 0 Then

        Exit Sub

    Else

            p_funLoadTemplate emn_TemplateBasedForms.Contacts, lngId

    End If

 

      Or from the bookmark component

    p_lng_frmSearchId = 0

   DoCmd.OpenForm "frmBookmarks", acNormal, , , , acDialog, 0 & "|" & c_Forms_frmContactsRecord

 

    If p_lng_frmSearchId = 0 Then

        Exit Sub

    Else

        p_funLoadTemplate emn_TemplateBasedForms.Contacts, p_lng_frmSearchId

    End If
